# dockggu-frontend
독꾸와 함께, 책 완독을 목표로 

## 서비스 소개
‘독꾸’는 ‘독서를 꾸준히’의 줄임말이며 핵심기능은 북커톤입니다. 북커톤이란 책과 마라톤의 합성어로 마라톤에 참가하는 듯이 모임에 참가하여 기간 안에 사람들과 함께 완독을 목표로 책을 꾸준히 읽어나가는 것을 의미합니다. 이 서비스를 통해 사람들이 책을 읽는 데 동기부여를 받을 수 있습니다.
